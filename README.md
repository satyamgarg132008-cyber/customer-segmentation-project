# Customer Segmentation Project

## Overview

This project performs customer segmentation using the K-Means Clustering algorithm. Customers are grouped based on their Annual Income and Spending Score to identify different purchasing behavior patterns.

## Objective

The main objective of this project is to segment customers into meaningful groups for better business understanding and targeted marketing strategies.

## Dataset

Dataset Used: Mall_Customers.csv

Features:

* CustomerID
* Gender
* Age
* Annual Income (k$)
* Spending Score (1-100)

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* Scikit-Learn
* Google Colab

## Methodology

1. Load the customer dataset.
2. Select Annual Income and Spending Score as features.
3. Use the Elbow Method to determine the optimal number of clusters.
4. Apply K-Means Clustering.
5. Visualize customer segments.
6. Analyze cluster characteristics.

## Results

The customers were divided into 5 clusters:

* High Income – High Spending
* High Income – Low Spending
* Low Income – High Spending
* Low Income – Low Spending
* Average Customers

## Output

The project generates:

* Dataset preview
* Elbow Method graph
* Customer Segmentation graph
* Cluster Summary table

## Conclusion

K-Means clustering successfully grouped customers based on their spending behavior and income levels. These segments can help businesses understand customer patterns and improve marketing strategies.
